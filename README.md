LimitlessLED
============

NodeJS control for LimitlessLED (MiLight, EasyBulb, ) wifi controller 3.0

## Install

Installling this is as easy as:

```bash
npm install LimitlessGEM
```

Or you can include it in your package.json to use it in your own application
